## 🚀 Tecnologias

Tecnologias utilizadas durante o projeto:

- [ReactJS](https://pt-br.reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [MirageJS](https://miragejs.com/)
- [Sass](https://sass-lang.com/)

## ⭐ Funcionalidades implantadas
- Refatorar uma página para listagem de filmes de acordo com gênero
- Uso de context do React

## 💻 Projeto

O WatchMen é uma plataforma de filmes, onde o usuário pode filtrar a listagem de filmes ao selecionar pelo gênero (terror, ação, comédia e entre outros) e treinar o ReactJS 😄. 

Este projeto é o desafio II oferecido pela Rocketseat referente ao módulo 1 da Trilha ReactJS.

## 😎 Iniciando 

📖 Para rodar o projeto localmente você precisará seguir algumas instruções.

- É necessário que o <a href="https://nodejs.org/en/">NodeJS</a> esteja instalado em sua máquina.
- É necessário utilizar <a href="https://classic.yarnpkg.com/en/">Yarn</a> ou <a href="https://www.npmjs.com/">Npm</a> para a instalação de pacotes.
- É rodar a Fake API com JSON Server utilizando yarn server.

### Instalação
```bash
# Clone o repositório
git clone https://github.com/larissadantier/desafio02-trilha-reactjs.git

# Instale todos os pacotes

- Npm
npm install

- Yarn
yarn install

```
